# ipl_data_analysis
IPL Data Analysis Dashboard (2008–2024)

This project analyzes Indian Premier League (IPL) data to uncover insigdhts about team performance, player statistics, and match trends.
The analysis is performed using Python with libraries like Pandas, Matplotlib, and Seaborn.

Key Objectives:
Analyze team performance across seasons
Identify top-performing batters and bowlers
Evaluate the impact of toss decisions
Study trends in matches and scoring patterns over time
Present insights in a unified dashboard

Dataset Used
matches.csv → Match-level data (teams, winners, toss, season, etc.)
deliveries.csv → Ball-by-ball data (runs, wickets, players)

 Data Preprocessing
Handled missing values and duplicates
Standardized team names:
Delhi Daredevils → Delhi Capitals
Kings XI Punjab → Punjab Kings
Filtered only active IPL teams
Removed irrelevant or inconsistent records


Key Insights
Some teams show consistently higher win rates, indicating strong long-term performance
Top batters maintain dominance across multiple seasons
Toss advantage is not decisive but situational
Average runs per match have increased over time → more aggressive gameplay
Number of matches varies by season due to league expansion
 Tech Stack
Python 
Pandas (Data Analysis)
Matplotlib & Seaborn (Visualization)

 How to Run?
Clone the repository

Install dependencies:

pip install pandas matplotlib seaborn

Run the notebook:

jupyter notebook
